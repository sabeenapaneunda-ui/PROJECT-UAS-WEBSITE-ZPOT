# PROJECT-UAS-WEBSITE-ZPOT
ZPOT (dibaca: Spot) adalah aplikasi web untuk eksplorasi konten berbasis kategori. Dilengkapi fitur autentikasi, halaman explore interaktif, dan manajemen profil pengguna. Dikembangkan sebagai proyek Pemrograman Web dengan alur sistem terstruktur dan fokus pada pengalaman pengguna.
